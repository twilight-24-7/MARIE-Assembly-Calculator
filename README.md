# MARIE-Assembly-Calculator
This program accepts user defined string input, extracts 2 operands and operator, then performs simple arithmetic operation,
then output to "output log" and "16x16 display"

To run the program, open any browser
(browser with specific extensions may not work well, such as "dark reader", or any configurations that resist fingerprinting):

Download / copy and upload / paste calculator.mas to MARIE.js intepreter: https://marie.js.org/ 

Click "Assemble" to compile program, then click "Run" to execute program 

You may adjust execution speed, click "step" / "microstep" to visualize how instructions are executed
